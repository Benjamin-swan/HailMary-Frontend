# 결제 진행 동의 안내

도화선 유료 콘텐츠 결제 전 아래 사항을 확인해 주세요.

## 1. 다시보기 이용 기간

구매하신 유료 결과지(정밀 리포트)는 **결제일로부터 30일** 동안 이메일로 받으신 조회 링크를 통해 반복 조회할 수 있어요. 30일 경과 후에는 결과지와 관련 입력 정보가 자동 파기되어 복구·재조회가 불가능합니다.

## 2. 청약철회 제한

도화선의 유료 결과지는 결제 즉시 인공지능을 통해 생성·제공되는 디지털 콘텐츠입니다.
「전자상거래 등에서의 소비자보호에 관한 법률」 제17조 제2항 단서에 따라 **다음의 경우 청약철회가 제한**됩니다.

- 이용자가 결과지를 이미 조회·열람한 경우
- 결과지가 결제 즉시 생성·제공되어 디지털 콘텐츠로서의 본질이 소비된 경우
- 결과의 복제·캡처·전달 등으로 재화의 가치가 현저히 감소한 경우

## 3. 환불 가능 케이스

다음의 경우에는 결과 조회 여부와 관계없이 환불을 진행합니다.

- 회사 측 시스템 오류로 결과지가 정상 제공되지 않거나, 동일·유사한 결과지의 재제공이 불가능한 경우
- 결제일로부터 30일 경과 전에 회사 귀책사유로 서비스가 중단되어 정상 조회가 불가능해진 경우
- 만 19세 미만 미성년자가 법정대리인의 동의 없이 결제한 경우(관련 법령 한도 내)

## 4. 결제 수단 보안

결제 수단(카드번호, 만료일, CVC 등) 정보는 PCI-DSS 보안 인증을 갖춘 결제대행사가 직접 수집·처리하며, 도화선은 해당 정보를 보관하지 않습니다.
